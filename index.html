🎧 Listen to Internet Radio, Straight from Your Terminal 🎶

A lightweight, terminal-based internet radio player built with Nim. 🚀

📥 Install for Unix 📥 Install for Windows

✨ Features

📦 Portable

Works seamlessly on Unix and Windows.

🎮 Easy to Use

Menu-driven interface for effortless navigation.

📝 Modifiable Stations

Edit JSON files to add or remove radio stations.

⚡ Lightweight

Minimal dependencies for fast performance.

🔧 Installation

🐧 Unix

nimble install pnimrp

🪟 Windows

nimble install pnimrp
